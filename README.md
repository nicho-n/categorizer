# Categorizer
_Very responsive!_<br><br>
The myspace-style social media site with categories - connecting people with esoteric interests.

## Build
`npm run build`

## Deploy 
`npm run start`

## Entry
`http://localhost:3000`

## Notes
less file needs to be compiled with `npm build` to see correct styling
